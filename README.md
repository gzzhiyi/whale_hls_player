# Whale HLS-Player

  whale-hls-player 是一个支持在PC、移动端实现 HTTP Live Streaming 的 JavaScript 库。

## Features

  - HLS视频流解析
  - 字幕文件解析

## Install

  ```bash
    git clone https://github.com/gzzhiyi/whale_hls_player.git
  ```

### npm install

  ```bash
    npm i whale-hls-player
  ```

### yarn install

  ```bash
    yarn add whale-hls-player
  ```
